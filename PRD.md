# PRD: 입지 선정 분석 툴 (Location Intelligence Tool)

**작성일:** 2026-03-27
**버전:** v0.2

---

## 1. 개요 (Overview)

### 1.1 제품 목표

소상공인 및 사업 기획자가 **데이터 기반**으로 최적 입지를 선정할 수 있도록 돕는 Python 기반 분석 툴.
주관적 감에 의존하던 입지 결정을 점수 모델·공간 분석으로 객관화한다.

- **분석 범위:** 전국 (시·도 단위 선택 가능, 기본값: 전국 전체)
- **격자 셀 크기:** 사용자 지정 가능 (기본값 500m, 선택지: 250m / 500m / 1km / 2km)
- **실행 환경:** CLI + Jupyter Notebook

### 1.2 핵심 수식

```
좋은 입지 = 수요(고객) ↑ + 경쟁 ↓ + 접근성 ↑

Score = (인구 * w1) + (유동인구 * w2) - (경쟁업체수 * w3) + (접근성 * w4)
```

가중치(`w1~w4`)는 업종별로 달리 설정 가능.

## 2. 문제 정의 (Problem Statement)

| 현재 문제 | 이 툴이 해결하는 것 |
|---|---|
| 입지 선정이 경험/감에 의존 | 데이터 기반 점수화로 객관화 |
| 여러 후보지 비교가 어려움 | 격자·반경 단위 일괄 비교 |
| 경쟁/수요 분석이 수작업 | 자동 집계 및 시각화 |
| 업종별 특성 반영 어려움 | 가중치 프리셋으로 업종 맞춤화 |

## 4. 기능 요구사항 (Functional Requirements)

### 4.1 데이터 수집 및 관리

| ID | 기능 | 우선순위 |
|---|---|---|
| D-01 | 통계청 인구 데이터 로드 (CSV / API) | Must |
| D-02 | 서울 열린데이터광장 유동인구 연동 | Should |
| D-03 | 경쟁업체 위치 데이터 로드 (CSV 또는 카카오맵 API) | Must |
| D-04 | 지하철·버스 정류장 위치 데이터 로드 | Should |
| D-05 | 카드 매출·소비 패턴 데이터 연동 | Could |

### 4.2 분석 기능

| ID | 기능 | 우선순위 |
|---|---|---|
| A-01 | **격자(Grid) 분석** — 지역을 N×N 셀로 분할, 셀별 지표 집계 / 셀 크기 파라미터로 가변 설정 | Must |
| A-02 | **점수화 모델(Scoring)** — 가중치 적용 종합 점수 산출 | Must |
| A-03 | **반경 분석(Buffer)** — 특정 좌표 기준 반경 내 지표 집계 / 반경 거리 파라미터로 가변 설정 | Must |
| A-04 | **클러스터링(Hotspot)** — 수요 밀집 / 경쟁 공백 지역 탐지 | Should |
| A-05 | 업종별 가중치 프리셋 (카페, 병원, 편의점 등) | Should |
| A-06 | 복수 후보지 일괄 비교 및 순위 산출 | Must |

### 4.3 시각화 및 출력

| ID | 기능 | 우선순위 |
|---|---|---|
| V-01 | 격자 히트맵 지도 출력 (folium / geopandas) | Must |
| V-02 | 반경 버퍼 원형 오버레이 지도 출력 | Must |
| V-03 | 후보지별 점수 바 차트 비교 | Should |
| V-04 | HTML 리포트 자동 생성 | Could |

## 5. 비기능 요구사항 (Non-Functional Requirements)

| 항목 | 요구사항 |
|---|---|
| 언어 | Python 3.10+ |
| 주요 라이브러리 | `geopandas`, `shapely`, `folium`, `pandas`, `scikit-learn`, `matplotlib` |
| 입력 형식 | CSV, GeoJSON, REST API |
| 실행 방식 | CLI + Jupyter Notebook 병행 지원 (웹 UI 미포함) |
| 좌표계 | EPSG:4326 (WGS84) 기본, 분석 시 EPSG:5179 (TM) 변환 |

## 7. 시스템 아키텍처 (안)

```
loc int/
├── PRD.md                  ← 이 문서
├── data/
│   ├── raw/                ← 원본 데이터 (CSV, SHP)
│   └── processed/          ← 전처리 완료 데이터
├── src/
│   ├── loader.py           ← 데이터 로드·전처리
│   ├── grid.py             ← 격자 생성·집계 (A-01)
│   ├── scoring.py          ← 점수 모델 (A-02)
│   ← buffer.py            ← 반경 분석 (A-03)
│   ├── cluster.py          ← 클러스터링 (A-04)
│   └── visualizer.py       ← 지도·차트 시각화 (V-01~V-03)
├── presets/
│   └── weights.yaml        ← 업종별 가중치 프리셋 (A-05)
├── notebooks/
│   └── demo.ipynb          ← 분석 데모 노트북
└── main.py                 ← CLI 진입점
```
